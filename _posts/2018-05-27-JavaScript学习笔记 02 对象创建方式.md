## JavaScript中获得一个对象的方法

-   对象直接量

    ```javascript
    var o = {
        name: "tom"
    };
    ```

<!--more-->

-   构造函数

    ```javascript
    var o = new Object();
    ```

    ​

-   使用`Object.create()`

    `var o = Object.create(null);`
